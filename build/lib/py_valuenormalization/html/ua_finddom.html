Cost Model Calibration

To help picking the best clustering algorithm parameter settings, you can answer a few question about the data values and collections of data values.

Using your answers, we create a cost model to estimate the time you will spend cleaning up each clustering of input data values. Then we pick the clustering algorithm parameter settings such that you need to spend the least time cleaning up the clusters generated by using this clustering algorithm.

During this step you see 6 clusters of values in the input dataset, one cluster at a time. For each cluster, first scan the values in the cluster from top to bottom and understand each value. While scanning and understanding, keep a count of the number of values referring to each entity represented in the cluster.

After you are done scanning the values, decide which entity is the dominating entity of the cluster; i.e. the entity with the most number of values referring to it. Then click on the the first value in the cluster which refers to the dominating entity. This would take you to the next cluster.

When you're ready to start, press "Start Finding Dominating Entities" button.
