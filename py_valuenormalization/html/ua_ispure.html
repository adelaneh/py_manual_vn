Cost Model Calibration

To help picking the best clustering algorithm parameter settings, you can answer a few question about the data values and collections of data values.

Using your answers, we create a cost model to estimate the time you will spend cleaning up each clustering of input data values. Then we pick the clustering algorithm parameter settings such that you need to spend the least time cleaning up the clusters generated by using this clustering algorithm.

During this step you see 3 clusters of values in the input dataset, one cluster at a time. For each cluster, first determine whether it is pure or not; i.e. whether all of the values in the cluster refer to the same real-world entity or not. Then click "Yes" if it is pure and click "No" if it is not.

When you're ready to start, press "Start Pure Cluster Detection" button.

First determine whether the following cluster is pure or not; i.e. whether all of the values in this cluster refer to the same real-world entity or not. Then click "Yes" if it is pure and click "No" if it is not.
